# hex-color

[![npm version](https://img.shields.io/npm/v/hex-color.svg)](https://www.npmjs.com/package/hex-color)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Hex color utilities — parse, convert, validate, generate

## Installation

```bash
npm install hex-color
```

## Functions

- `parseHex` — Parse hex color to RGB object
- `rgbToHex` — Convert RGB values to hex string
- `isValidHex` — Validate hex color string
- `randomHex` — Generate random hex color
